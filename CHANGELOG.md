# 📜 HeadStart Changelog

All Notable Changes to the **HeadStart E-Platform** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/0.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [0.1.0] - April 09, 2026

### ✨ Added

- **Security Audit Protocol** : Created `documentation/ci-cd/SECURITY_AUDIT_PROTOCOL.md` to define Profile Lock & SASM Audit Gates.
- **CodeCov Integration** : Established Secure Coverage Reporting using `CODECOV_TOKEN` & full Git History Tracking.
- **Public Registry Hosting** : Configured GitHub Container Registry (GHCR) with specific **write** Permissions for Automated Image Hosting.

### 🔄 Changed

- **Rebranding** : Completed the Transition from PEND Boilerplate to HeadStart across all root & Module-Level `README.md` & Documentation Files.
- **Design System** : Formalized **Montserrat** (Headings) & **Inter** (Body) Typography in the Frontend Specification.
- **Color Palette** : Updated Brand colors to **Royal Purple (`#790BBE`)** & **HeadStart Gold (`#FEE054`)**.
- **Contributing Guidelines** : Updated `CONTRIBUTING.md` with the mandatory Backup Branch Protocol & Tag Branching Strategy.
- **Security Policy** : Revised `SECURITY.md` to reflect Production Reporting Channels & specialized Protection Features.
- **CI / CD Strategy** : Transitioned from a "Cost-Optimized" (Limited) Testing Strategy to a "High-Integrity" Strategy suitable for Public Repositories.
- **Linting Architecture** : Migrated Frontend Module to **ESLint 9 Flat Configuration**, integrating `typescript-eslint` & `@eslint/js`.
- **Dependency Management** : Optimized `package-lock.json` Resolution for `@standard-schema/spec` & ReactJS 19 Compatibility.

### 🛡️ Fixed

- **Documentation Mismatch** : Resolved conflicting Boilerplate References in License & Maintenance Files.
- **Link Integrity** : Updated GitHub Badges & Repository URLs to point to `corebit-bd/headstart`.
- **Security Vulnerability** : Patched `node-forge` to `v1.4.0` in the **Mobile** Module to address Critical Security Advisories.
- **Continuous Deployment (CD) Permission Errors** : Resolved `403 Forbidden` Registry Errors by implementing dedicated Registry Tokens & enhanced Job Permissions.
- **Linting Noise** : Suppressed ReactJS-specific Linting Rules for Storybook Configuration Files to improve Developer Experience (DevEx).

### ✨ Initialized

- **Project Scaffolding** : Cloned the PEND (PostgreSQL-Expo-NextJS-Django) Boilerplate as the Foundation for HeadStart.
- **CI/CD Baseline** : Established the initial 7-Gate Quality Pipeline & Layered Caching Strategy.
